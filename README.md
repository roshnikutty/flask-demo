# Run this app

1. create a vritual env, example .venv and source it
```python3 -m venv .venv```
```source .venv/bin/activate```

2. Then run app.py
```cd flask-tutorial```
```python app.py```


3. Navigate to home page or pass in a product id \
    `http://localhost:5000` \
    `http://localhost:5000/product/15`
